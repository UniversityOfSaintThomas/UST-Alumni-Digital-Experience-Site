## Accessibility — Color Contrast

Source: [UST Brand Hub — Color Contrast](https://one.stthomas.edu/sites/brand-hub/SitePageModern/254290/color-contrast)

> **Use only the accessible combinations listed below.** All combinations have been tested to meet **WCAG AA** standard.  
> **"Large Text Only"** means the pairing is only valid when text is **14pt Bold** or **18pt+ Regular**.

### WCAG AA Requirements (for reference)
| Text Type | Minimum Contrast Ratio |
|-----------|----------------------|
| Normal text (< 18pt / < 14pt bold) | **4.5:1** |
| Large text (≥ 18pt, or ≥ 14pt bold) | **3:1** |
| UI components & graphical objects | **3:1** |

---

### Pairings to Avoid
- **Emerald Green text on White** — large text only; do not use for body copy or normal-weight UI labels
- **Cool Gray on Dark Purple** — large text only
- **Black on Medium or Bright Purple** — large text only
- **Tints (any)** — tints are not included in approved contrast pairs; do not use as text colors

---

## Usage Guidelines

- **Dark Purple + Medium Purple** should dominate any UST design
- **Bright Purple** is an accent — use sparingly alongside the primaries
- **Emerald Green** reflects the heritage of founder John Ireland; use as an accent with strong accessibility contrast
- **Cool Gray** works for text, borders, and neutral backgrounds
- **Cool Gray @ 10% (`#f3f3f3`)** is recommended as a default page background
- Tints are intended for **data visualization** (charts, graphs) and **background fills** — not as replacements for the primary palette in UI chrome
